You need 1-3 screenshots showing the extension in action:

```

2. Go to https://myprize.us
3. Open the chat
4. Show the 🎰 button in the chat input
5. Click it to show the emoji popup
```

- Recommended: 1280×800

### 2. Go to Chrome Web Store (1 minute)

URL: https://chrome.google.com/webstore/devconsole

**Note:** First-time publishers need to pay a one-time $5 developer registration fee.

### 3. Upload Extension (2 minutes)

1. Click **"Add new item"**
2. Upload: `/home/aiuser/projects/myprize-emoji-picker/myprize-emoji-picker.zip`
3. Wait for upload to complete
