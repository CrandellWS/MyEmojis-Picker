🚀 Install

Up in 60 seconds.

No Chrome Web Store account needed. Load it directly from the downloaded folder.

1

Download the extension

Click the Download Extension button above and save the zip file. Unzip it anywhere on your computer.

2

Open Chrome Extensions

In Chrome, go to chrome://extensions and turn on Developer mode (toggle in the top right).

3

Load the folder

Click Load unpacked and select the unzipped folder. The extension appears instantly.

4

Head to MyPrize.us

Open the chat. You'll see a 🎰 button in the chat input. Click it and start reacting.

5

Updating the extension

When a new version is released, replace the folder contents and click the refresh icon on the extension card at chrome://extensions. Done.
